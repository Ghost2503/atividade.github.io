perguntas de HTML:

  • O que é HTML? Qual a sua função principal no desenvolvimento web? Por que não é considerada uma linguagem de programação?
  • Estrutura Básica de um Documento HTML: Quais são as tags essenciais (DOCTYPE html>, html, head, body) e qual a função de cada uma?
  • Tags de Conteúdo: Pesquise sobre as tags h1 (título principal), p (parágrafo) e img (imagem). Quais são os atributos importantes para a tag img (ex: src, alt) e para que servem?

respostas:

  • HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar e organizar o conteúdo na web. Sua função principal no desenvolvimento web é definir a estrutura das páginas, como textos, imagens, links e outros elementos, permitindo que os navegadores exibam corretamente o conteúdo para os usuários. HTML não é considerada uma linguagem de programação porque não possui lógica de programação, como condições, loops ou manipulação de dados; ela apenas marca e organiza o conteúdo, sem executar operações ou cálculos.
  • HTML é uma linguagem de marcação que estrutura o conteúdo da web. Não é linguagem de programação porque não executa lógica. A estrutura básica inclui:

    • DOCTYPE HTML: define o tipo do documento.

    HTML: raiz do documento.

    head: metadados e configurações.

    body: conteúdo visível da página.

  • As tags HTML H1, P e IMG têm funções essenciais na estrutura de uma página web. A tag H1 é usada para o título principal, destacando o conteúdo mais importante e ajudando no SEO. A tag P define parágrafos de texto, organizando o conteúdo para melhor leitura. A tag IMG insere imagens e possui atributos importantes, como o src, que indica o caminho da imagem, e o alt, que fornece uma descrição alternativa para acessibilidade e SEO. Outros atributos relevantes da IMG incluem title (texto explicativo), width e height (dimensões da imagem), loading (controle de carregamento para melhorar a performance) e srcset (para imagens responsivas). Esses elementos são fundamentais para criar páginas web acessíveis, organizadas e otimizadas.

Perguntas de CSS:

  • O que é CSS? Qual a sua função principal em relação ao HTML? Por que é importante para o desenvolvimento web? Por que não é considerada uma linguagem de programação?
  • Como incluir CSS no HTML: Pesquise sobre a tag style dentro do head e como ela é usada para escrever CSS diretamente no HTML
  • Seletores e Propriedades CSS: O que são seletores (ex: h1, p, img) e como eles são usados para aplicar estilos? Pesquise sobre as propriedades color, font-family, font-size, text-align, border e background-color

Respostas:

  • CSS, que significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para definir a apresentação visual de documentos estruturados em HTML
  • Incluir CSS diretamente no HTML usando a tag style dentro do head é conhecido como CSS Interno ou Incorporado. Essa técnica é ideal para aplicar estilos únicos a uma única página, sem a necessidade de criar um arquivo .css separado.
  • Seletores CSS são padrões utilizados para "selecionar" ou encontrar elementos HTML em uma página, permitindo que propriedades de estilo sejam aplicadas a eles. Eles formam a base da estrutura de uma regra CSS: seletor { propriedade: valor; }
